# Meal Explorer

Meal Explorer is a website that shows a random meal from a REST API.

## Features

- Get a random meal
- Show the meal name
- Show an image of the meal
- Show short cooking instructions

## Built with

- HTML
- CSS
- JavaScript
- Vitest
- Playwright

## Tests

- Unit tests for JavaScript functions
- DOM test
- Integration test for REST API
- End to End test with Playwright

## How to run

Open `index.html` with Live Server.

## Run tests

Vitest:

```bash
npm test
```

Playwright:

```bash
npx playwright test
```
